# crud--ddd-test

1. Install dependencies

```bash
composer install
```

2. Copy .env.example and generate app key

```bash
cp .env.example .env && php artisan key:generate
```

3. Migrate the database

```bash
php artisan migrate
```

4. Run the application

```bash
php artisan serve
```
